# Build tool

- Vite react powered project
- Vite is a build tool that aims to provide a faster and leaner development experience for modern web projects. see more (https://vite.dev/)

# Library

- lucide-react (icon)
- Tailwindcss (css)
- React-router-dom (page navigation)
- Axios
- React-validation
